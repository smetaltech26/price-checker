# 🏷️ ระบบเช็คราคาสินค้าและราคาวัตถุดิบ (Price & RM Price Checker)

สวัสดีค่ะพี่ต้น! นี่คือซอร์สโค้ดสำหรับระบบเช็คราคาสินค้าและราคาวัตถุดิบที่พี่ต้นต้องการค่ะ แอปพลิเคชันนี้ออกแบบมาให้โฮสต์บน **GitHub Pages** ได้ฟรี 100% และเชื่อมต่อข้อมูลจาก Google Sheets ได้โดยตรงค่ะ

แอปนี้รองรับการทำงานทั้งบนคอมพิวเตอร์และมือถือ มีการติดตั้งระบบ **Offline Cache** ช่วยให้โหลดเว็บได้รวดเร็วทันใจ พร้อมรองรับการทำงานออฟไลน์ และเมื่อสแกน QR Code สำเร็จจะมี **เสียงบี๊บ** แจ้งเตือนเสมือนใช้เครื่องสแกนระดับมืออาชีพเลยค่ะ! 🔊✨

## 🛠️ ขั้นตอนการเชื่อมต่อ Google Sheets

พี่ต้นสามารถเลือกเชื่อมต่อ Google Sheets ได้ 2 วิธีตามต้องการค่ะ (สามารถสลับได้ผ่านปุ่มตั้งค่าฟันเฟือง ⚙️ ในแอป):

### วิธีที่ 1: การใช้ Google Apps Script Web App API (แนะนำ ⭐)
วิธีนี้มีจุดเด่นคือ **ข้อมูลมีความปลอดภัยสูง** เพราะไม่ต้องเปิดไฟล์ Google Sheets ของพี่ต้นเป็นสาธารณะค่ะ

1. เปิดไฟล์ Google Sheets ของพี่ต้น
2. ไปที่เมนู **"ส่วนขยาย" (Extensions)** -> **"Apps Script"**
3. คัดลอกโค้ดทั้งหมดที่อยู่ในไฟล์ [gas-code.js](file:///E:/Project%20check%20price/gas-code.js) ของโปรเจกต์นี้ ไปวางทับในเครื่องมือแก้ไข Apps Script
4. กดบันทึก (ไอคอนแผ่นดิสก์)
5. กดปุ่ม **"การทำให้ใช้งานได้" (Deploy)** ที่มุมขวาบน -> เลือก **"การทำให้ใช้งานได้ใหม่" (New deployment)**
6. คลิกไอคอนรูปฟันเฟืองข้างคำว่า "เลือกประเภท" แล้วเลือก **"เว็บแอป" (Web app)**
7. ตั้งค่าการ deploy ดังนี้:
   *   **เรียกใช้เป็น (Execute as)**: เลือก "ฉัน" (อีเมล Google Sheets ของพี่ต้น)
   *   **ผู้มีสิทธิ์เข้าถึง (Who has access)**: เลือก **"ทุกคน" (Anyone)**
8. กดปุ่ม **"ทำให้ใช้งานได้" (Deploy)** จากนั้นกดให้สิทธิ์การเข้าถึง (Authorize access) และยืนยันความปลอดภัยให้เรียบร้อยค่ะ
9. คัดลอก **URL ของเว็บแอป (Web App URL)** ที่ยาวๆ (ลงท้ายด้วย `/exec`) นำไปเปิดแอปเช็คราคา กดรูปฟันเฟือง ⚙️ ด้านบน เลือกโหมด "Apps Script API" แล้ววาง URL นี้ลงไปเพื่อใช้งานได้ทันทีค่ะ!

### วิธีที่ 2: การใช้ Public Sheet URL (ใช้งานแบบรวดเร็ว)
วิธีนี้ง่ายและรวดเร็ว แต่พี่ต้นจำเป็นต้องเปิดแชร์ไฟล์ Google Sheets เป็นสาธารณะค่ะ

1. เปิดไฟล์ Google Sheets ของพี่ต้น
2. กดปุ่ม **"แชร์" (Share)** ที่มุมขวาบน
3. ตรงการเข้าถึงทั่วไป (General access) เปลี่ยนจาก "จำกัด" (Restricted) เป็น **"ทุกคนที่มีลิงก์" (Anyone with the link)** โดยกำหนดสิทธิ์เป็น **"ผู้มีสิทธิ์อ่าน" (Viewer)**
4. คัดลอก **Spreadsheet ID** จาก URL ของหน้าชีต ตัวอย่างเช่น:
   *   จากลิงก์: `https://docs.google.com/spreadsheets/d/1Aq1ZvwqKVKDQGIynEILrFIEe6A-sUqFk9Ztxm6SrNPo/edit...`
   *   Spreadsheet ID คือ: `1Aq1ZvwqKVKDQGIynEILrFIEe6A-sUqFk9Ztxm6SrNPo`
5. เปิดแอปเช็คราคา กดปุ่มฟันเฟือง ⚙️ ด้านบน เลือกโหมด "Public Sheet URL" วาง Spreadsheet ID ลงไป แล้วกดบันทึกใช้งานได้เลยค่ะ

## 🚀 ขั้นตอนการสร้าง GitHub Repository และอัปโหลดไฟล์ (New Project)

หากพี่ต้นต้องการนำไฟล์โปรเจกต์นี้ขึ้นไปฝากไว้บน GitHub ให้ทำตามขั้นตอนดังนี้ได้เลยค่ะ:

### 1. เข้าสู่ระบบ GitHub
1. เปิดเว็บไซต์ [GitHub.com](https://github.com/) และลงชื่อเข้าใช้ด้วยบัญชีของพี่ต้น (`smetaltech26`)
2. กดเครื่องหมายบวก **"+"** ที่มุมขวาบน -> เลือก **"New repository"**
3. ตั้งชื่อ Repository เช่น `price-checker`
4. ตั้งสถานะเป็น **Public** (หากต้องการให้ใช้งาน GitHub Pages ได้ฟรีแบบโฮสต์หน้าเว็บ) หรือ **Private** (หากต้องการเก็บเป็นความลับ)
5. **ห้ามติ๊ก** เลือก "Add a README file" หรืออื่นๆ ให้ปล่อยว่างทั้งหมดค่ะ
6. กดปุ่ม **"Create repository"**
7. ระบบจะแสดงหน้าคำสั่งสำหรับ Push โค้ด ค้างหน้านี้ไว้ก่อนนะคะ

### 2. รันคำสั่ง Git ในคอมพิวเตอร์ของพี่ต้น
เปิดโปรแกรม Terminal หรือ Git Bash ในเครื่องคอมพิวเตอร์ของพี่ต้น แล้วเข้าสู่โฟลเดอร์โครงการ จากนั้นรันคำสั่งเหล่านี้ทีละบรรทัดค่ะ:

```bash
# 1. เริ่มต้นใช้งาน Git ในโฟลเดอร์นี้
git init

# 2. เพิ่มไฟล์ทั้งหมดเข้าคิวเตรียม Commit
git add .

# 3. บันทึกประวัติเวอร์ชันแรก
git commit -m "First commit: Initialize Price Checker Web App"

# 4. ตั้งชื่อสาขาหลักเป็น main
git branch -M main

# 5. เชื่อมต่อ Git ในเครื่องเข้ากับ GitHub (ก๊อปปี้บรรทัดนี้จากหน้าเว็บ GitHub ของพี่ต้นจะชัวร์ที่สุดค่ะ)
git remote add origin https://github.com/smetaltech26/price-checker.git

# 6. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub
git push -u origin main
```

## 🌐 การเปิดใช้งานเว็บแอปผ่าน GitHub Pages (เพื่อให้สแกนกล้องบนมือถือได้)

เนื่องจากเว็บเบราว์เซอร์บนมือถือจะอนุญาตให้ใช้กล้องสแกน QR Code ได้เฉพาะเว็บที่เป็น **HTTPS (มีความปลอดภัย)** เท่านั้น การนำเว็บขึ้นโฮสต์ฟรีบน GitHub Pages จึงตอบโจทย์ที่สุดค่ะ:

1. ในหน้า Repository `price-checker` บนเว็บ GitHub ให้ไปที่เมนู **"Settings"** (แท็บด้านบนรูปฟันเฟือง)
2. เลือกเมนู **"Pages"** ในคอลัมน์ด้านซ้าย
3. ในหัวข้อ **Build and deployment**:
   *   ตรงคำว่า **Source**: ปล่อยเป็น "Deploy from a branch"
   *   ตรงคำว่า **Branch**: เปลี่ยนจาก `None` เป็น `main` (และเลือกโฟลเดอร์ `/ (root)`)
4. กดปุ่ม **"Save"**
5. รอประมาณ 1-2 นาที จากนั้นรีเฟรชหน้าเว็บ พี่ต้นจะได้รับลิงก์เว็บ HTTPS สำหรับเข้าใช้งานจากมือถือ เช่น:
   `https://smetaltech26.github.io/price-checker/`
6. เปิดลิงก์นี้จากมือถือหรือส่งต่อให้ผู้ใช้งานคนอื่นๆ สแกนกล้องสแกนราคาได้ทันทีค่ะ!
